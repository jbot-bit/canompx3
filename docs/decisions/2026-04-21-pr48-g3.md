N̂ = K / (1 + (K−1)·ρ̄) = 3 / (1 + 2·ρ̄)

| ρ̄ | N̂ |
|---|---|
| 0.3 | 1.875 |
| 0.5 | 1.500 |
| 0.7 | 1.250 |

Expected max-SR under null per Eq. 10 (finite-sample IID normals):
E[max SR_K] ≈ ((1 − γ_em)·Φ⁻¹(1 − 1/N̂) + γ_em·Φ⁻¹(1 − 1/(N̂·e))) · (1/√T)

where γ_em = 0.5772... is the Euler-Mascheroni constant.

DSR formula per Eq. 2:
DSR = Φ( ((SR − SR_0) · √(T−1)) / √(1 − γ₃·SR + ((γ₄−1)/4)·SR²) )

Data: unfiltered baseline pnl_r for each instrument at E2 CB1 RR1.5 5m, IS window (trading_day < 2026-01-01). Read-only DuckDB via `DUCKDB_PATH` override. Wall-clock 0.4s.

## Comparison to prior CONDITIONAL_PASS rationale

The prior CONDITIONAL_PASS rationale ("K=3 structural argument clears DSR qualitatively ... effect sizes of |t|>7 imply DSR >> 0.95") was wrong because it conflated the rank-effect t-statistic (which IS extreme) with the per-trade Sharpe ratio (which is small or negative). DSR is computed on SR, not on rank-regression t. The structural argument held for the rank-effect reality; it did NOT hold for deployment-form Sharpe.

**Lesson:** t-stat of a rank-regression effect and Sharpe of the deployment return stream are different animals. DSR targets the deployment Sharpe. Stretching structural arguments across that gap produced the CONDITIONAL_PASS optimism. F7 remediation corrects it with numerical evidence.
