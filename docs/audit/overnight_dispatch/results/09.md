**KEY RESULT FOR ROWS 01/08:** `_emergency_flatten` sends REAL market close orders to the broker (3 retry attempts, exponential backoff `2**attempt`), and on full failure does `log.critical + _notify + "MANUAL CLOSE REQUIRED"` (`:3085-3093`) — NOT silently swallowed. The EOD backstop `_kill_switch_positions_confirmed_flat` re-queries broker truth and fails-closed on broker errors. **So the rows 01/08 fix (call both methods) is structurally sound — IF BOTH are called together.**

**[MED] `_fire_kill_switch` is flag-only by design.** `:1067-1076`, docstring `:1070`: "Does NOT flatten — flattening is a separate `_emergency_flatten()` call." Any call site invoking only the kill-switch leaves the position OPEN — exactly the row-01 P0 bug. Fix for 01/08 MUST call both.

**[MED] `on_exit_filled` marks local state FLAT on submit() success, before fill confirmation.** `:3071-3076`, `position_tracker.py:202-205`: after `order_router.submit()` returns, `del self._positions[strategy_id]`. Submit = acceptance, not fill. If the close is rejected/partial post-submit, local tracker says FLAT while broker still holds the position; only EOD re-query catches it. FIX: mark `PENDING_EXIT` on submit, call `on_exit_filled` only on real fill callback.

**[MED] No-event-loop flatten path silently drops the order.** `:1290-1303`: feed-dead path does `loop.create_task(_emergency_flatten())`; if no running loop (post_session context), except block only `log.critical + _notify` — **flatten SKIPPED entirely**, no retry. Operator alerted but exposure persists until manual close. FIX: schedule via `asyncio.ensure_future`/queue for the post_session loop rather than dropping.

**[LOW] Notification-channel failure not guarded** (fail-open) on the retry-exhaustion alert path.

**SEVERITY:** flatten works on the happy path; the submit-before-fill lie and no-loop drop are real silent-exposure windows. The rows 01/08 fixes depend on this method — they are valid, but inherit these 2 gaps.

**Decision: ACCEPT_WITH_RISK** — flatten is genuine; fix the 2 MED gaps alongside the rows 01/08 naked-position fix so the whole chain is sound.
