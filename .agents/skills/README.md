# Codex Repo Skills

These are thin discovery wrappers for the official Codex repo skill path.

- Official Codex discovery scans `.agents/skills/`.
- Canonical repo-local Codex skill sources live in `.codex/skills/`.
- Each wrapper `SKILL.md` points back to the canonical source so there is still
  only one maintained Codex instruction layer.
- Use `canompx3-claude-parity` when a Claude workflow exists but Codex has no
  dedicated wrapper yet.
